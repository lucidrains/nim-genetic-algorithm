# Genetic Algorithm in Nim

a simple genetic algorithm written in <a href="https://nim-lang.org/">Nim</a>

## Dependencies

```bash
$ nimble install malebolgia
```

## Usage

```bash
$ nim c -d:ThreadPoolSize=8 -d:release -r ga.nim
```
